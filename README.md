# NoChat

Stop specific people from talking 

## Installation

[Download Plugin](https://github.com/spiretf/nochat/raw/main/plugin/nochat.smx)

## Usage

- Enable the plugin
- Set the users to block: `rcon sm_nochat_blocklist 76561198024494988,76561198076020012`
